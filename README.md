# Bến Chill Garden Website

Gói này dùng được cho GitHub Pages.

## Cách upload nhanh

1. Tạo repository mới trên GitHub.
2. Upload toàn bộ nội dung trong thư mục này, giữ nguyên `index.html`, `assets/`, `ben-chill-logo.png` và `menu-ben-chill-garden.pdf`.
3. Vào `Settings` -> `Pages`.
4. Chọn `Deploy from a branch`, branch `main`, folder `/root`.
5. Lưu lại và chờ GitHub Pages tạo link online.

## Thay menu PDF

Thay file `menu-ben-chill-garden.pdf` bằng file menu chính thức cùng tên.

## Thay video thật

Hiện chưa có file video trong thư mục nguồn. Khi có video MP4, thêm vào `assets/` và thay phần tab Video trong `index.html` bằng thẻ `<video controls>`.
